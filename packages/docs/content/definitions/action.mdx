## Action

A function that receives the machine's current `data`, `payload`, and `result` as arguments, and that may mutate the machine's [data](#data).

```jsx
const machine = createStateDesign({
  data: { count: 0 },
  on: {
    PLUS_CLICKED: "incrementCount",
  },
  actions: {
    // highlight-start
    incrementCount(data, payload, result) {
      data.count++
    },
    // highlight-end
  },
})
```
